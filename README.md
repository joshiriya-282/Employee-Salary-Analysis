# Employee-Salary-Analysis
# Employee Salary Analysis using Excel

## Project Overview
This project analyzes employee salary data using Microsoft Excel.

## Dataset
- 1000 employee records
- Age
- Experience
- Education Level
- Job Role
- Salary

## Analysis Performed
- Data Cleaning
- Salary Classification
- Pivot Tables
- Job Role Analysis
- Education Analysis
- Experience vs Salary Analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Formulas

## Key Insights
- Salary increases with experience.
- Education level affects salary.
- Different job roles have different salary ranges.
